Mount the Sensor radio module on the meter body as shown in video. Be sure that the LED-status indicator flashes once shortly immediately after mounting and then it flashes several times more in ~20 sec. Herewith activation of the Sensor radio module is done automatically. If you have already installed the sensor on meter, remove it and wait 2 minutes before next try. Please make sure you have followed all instructions carefully and press install button. If you use a router, check its power and Internet connection status.
